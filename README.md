# koishi-plugin-bridge-discord-qq
让discord和qq的消息互通

目前为自用插件，现有功能应该能正常使用，如果你需要了解使用这个插件的方法可以联系我

npmjs上的插件为最新版本，如果Github的版本和npmjs的版本不同步，以npmjs为准 ~~（其实就是懒得同步）~~

# 功能列表
1. QQ
    - [x] 消息同步
    - [x] 回复消息同步
    - [ ] @消息同步
    - [x] 图片,gif同步
    - [x] 表情同步
    - [x] 文件同步
    - [x] 视频同步
    - [x] 合并转发消息同步 
2. Discord
    - [x] 消息同步
    - [x] 回复消息同步
    - [ ] @消息同步
    - [x] 图片,gif同步
    - [x] 表情同步
    - [x] 文件同步
    - [x] 视频同步
