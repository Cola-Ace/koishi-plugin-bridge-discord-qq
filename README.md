# koishi-plugin-bridge-discord-qq
让discord和qq的消息互通

目前为自用插件，如果你实在需要使用这个插件请联系我

npmjs上的插件为最新版本，如果Github的版本和npmjs的版本不同步，以npmjs为准

# 功能列表
1. QQ
    - [x] 消息同步
    - [x] 回复消息同步
    - [ ] @消息同步
    - [x] 图片,gif同步
    - [x] 表情同步
    - [x] 文件同步
    - [x] 视频同步
2. Discord
    - [x] 消息同步
    - [x] 回复消息同步
    - [ ] @消息同步
    - [x] 图片,gif同步
    - [x] 表情同步
    - [ ] 文件同步
    - [x] 视频同步
